# Attendify App

A mobile-based Smart Attendance System developed using Flutter and Firebase for efficient attendance management.

## Features
- User Authentication
- Mark Attendance
- Attendance Tracking
- Attendance History
- Firebase Integration
- Real-Time Data Storage
- Clean UI Design
- Mobile Responsive Interface

## Tech Stack
- Flutter
- Dart
- Firebase
- Android Studio
